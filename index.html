Imgurify Social App

Developed a personal project, replicating the functionality of Imgur, a widely used images hosting and sharing platform. This project is a React-based web application that emulates some of the core features of Imgur, a popular online image hosting and sharing platform. It allows users to browse, upload, like, and comment on images.

Imgurify Social App API

As part of the Imgurify project, I developed a robust API supporting the core functionalities of the application. Leveraging Node.js, Express and Mongoose, I crafted an efficient backend system that seamlessly integrates with the React-based frontend. This backend powers the browsing, uploading, liking, and commenting features, offering users a seamless experience akin to the Imgur platform.
