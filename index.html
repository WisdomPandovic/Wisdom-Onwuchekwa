Mini-Commerce App

Mini-Commerce is a sleek, full-stack e-commerce application built with Next.js 14 and TypeScript. It features a dynamic product catalog, detailed product views, search, wishlist, shopping cart, and a full checkout flow with order confirmation. I integrated Zustand for global state management, React Query for data fetching and caching, and styled it using Tailwind CSS. The project is optimized for performance, accessibility, and SEO — showcasing a production-ready approach to modern web development.
